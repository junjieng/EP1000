About Circuit Design

In Circuit deign we will be using Thinkercad to create our circuit and after we have create our own circuit we will be able to run on its own or use switch to control For Audrino we will also be using Thinkercad and Audrino Software to type the coding and after we have type in everything in the Audrino we will have to transfer the coding to the thinkercad to run the circuit.
